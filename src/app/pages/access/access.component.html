Instalacja kontroli dostępu

W naszej ofercie znajduje się wykonywanie instalacji KD. Realizujemy następujące elementy składowe systemu:

  • done Montaż czytników (klawiatury kodowe, czytniki kart magnetycznych, karty RFID)
  • done Kontrolery dostępu (klawiatury szyfrowe)
  • doneInterfejsy
  • done Urządzenia blokujące (urządzenia wyjściowe realizujące udostępnianie stref chronionych, czyli rygle elektromagnetyczne, zamki elektryczne, bramki, szlabany itp.)
  • done Akcesoria (klawiatury, czujniki, przyciski wyjścia, wyświetlacze)

Urządzenia wejściowe systemu KD wykonujemy w postaci pojedynczych oddzielnych urządzeń lub jako urządzenia zespolone zawierające w jednej obudowie kilka elementów składowych np. czytniki z klawiaturą, czytniki z wyświetlaczem, czytniki z wbudowanym kontrolerem, wyświetlaczem i interfejsem, zwory elektromagnetyczne z wbudowanym czujnikiem otwarcia drzwi itp.
